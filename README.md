# TFG-OSRM-Base

## Trabajo Fin de Grado

**Optimización de rutas de operadores en sistemas logísticos**

### Descripción

Este repositorio contiene los archivos base necesarios para la construcción del servidor OSRM empleado (v5.27.1) en el desarrollo del Trabajo Fin de Grado.

Los recursos incluidos permiten generar el servicio de enrutamiento utilizado por la aplicación. No se incorporan los archivos generados automáticamente durante el proceso de preprocesado, ya que estos pueden obtenerse mediante los scripts y procedimientos implementados en el proyecto.

### Contenido

* Archivos base del servidor OSRM.
* Librerías necesarias para los perfiles de desplazamiento integrados.
